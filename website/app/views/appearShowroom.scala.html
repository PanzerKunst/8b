Appear Showroom website

www.appearshowroom.com is a website showcasing the features of the Appear IQ product. Graphic design was done by Tom Knowles, and development was carried out in-house. It contains some advanced features, such as letting the user interact with pure JavaScript webapps in an iframe. These are mobile JS webapps developed with Appear IQ.

Work

I coded the backend of the website (Play Framework Scala & CouchDB database), and some of the frontend (based on the Backbone.js framework).
