Appear Showroom website

www.appearshowroom.com is a website showcasing the features of the Appear IQ product. Graphic design was done by Tom Knowles. It contains some advanced features, such as letting the user interact with pure JavaScript webapps in an iframe. These are mobile JS webapps developed with Appear IQ.

Work

I coded the backend of the website (Play Framework Scala & CouchDB database), and some of the frontend (based on the Backbone.js framework).

Screenshots

Homepage

Page listing the mobile webapps available in the showroom. Those apps are hybrid mobile webapps (pure JavaScript).

After choosing a webapp, you can configure it several ways: such as its layout and data on the backend. The right section contains an iframe where the JavaScript webapp is executed. The app can be tested in real time!

It's so fun to play with the webapp in the iframe that we added a real-size preview.

On this page, it's possible to select which apps are to be published to your mobile device.
