Site Diary tablet webapp

Site Diary is a cross-platform single-page webapp for iPads and Android tablets. It was created by Appear for construction group Costain. The app has access to native features such as the camera via the Cordova API, and it uses Appear IQ to synchronize with the backend.

Work

I did part of the developement of this JavaScript webapp, based on the SpineJS MVC framework.
